**CodeLingo** est une application web interactive d'apprentissage du code inspirée par l'approche ludique de Duolingo. Apprends le HTML, le Python, le JavaScript et le Go à travers des exercices rapides, gagne de l'XP et forge ton profil de développeur "Titan".

---

## ✨ Fonctionnalités Clés

### 🎯 Système de Maîtrise (Mastery)
Fini l'échec définitif ! Si tu te trompes, la question est mise de côté et revient à la fin de la leçon sous forme d'**Ancienne Erreur**. Tu ne valides pas le niveau tant que tu n'as pas tout compris.

### 👤 Profil Titan Dynamique
* **Stats en temps réel** : Suis ton XP, tes jours de série (streak) et tes badges.
* **Succès débloqués** : Gagne des trophées comme "Génie en Herbe" ou "Premier Pas".
* **Progression par niveau** : Ton titre évolue en fonction de tes points accumulés.

### 📚 Bibliothèque Massive (300+ Leçons)
* **Multi-langages** : HTML5, CSS3, Python, JavaScript (ES6+), et Go.
* **Exercices variés** : QCM tactiques et complétion de code (Input) en conditions réelles.
* **Explications sémantiques** : Chaque réponse (juste ou fausse) est accompagnée d'une explication technique pour progresser plus vite.

---

## 🚀 Installation Rapide

Ce projet est **Single Page Application (SPA)** sans backend complexe. Il suffit d'un navigateur !
